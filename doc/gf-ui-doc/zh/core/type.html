# isValidPassword

  • 是否为合法密码(由8~30位数字/字母/下划线组成)
  • 参数(optinons)
    • 默认值：{ min:8, max:30 }
Type.isValidPassword("gf-ui") // false
Type.isValidPassword("gf-ui/core@1.0.0") // true
Type.isValidPassword('gf-ui/core@1.0.0', {min: 5, max: 10}) // false
